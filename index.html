Speed Latest Video Navigator

This Chrome extension automatically launches YouTube, finds IShowSpeed's official channel, and opens the most recent upload in the current tab. The popup is keyboard accessible, provides live status updates, and displays a confirmation message when navigation succeeds.

Install Locally

  1. Download the extension package using the button below.
  2. Unzip the archive to a convenient folder.
  3. Open chrome://extensions in Google Chrome.
  4. Enable Developer mode in the top-right corner.
  5. Click Load unpacked and select the unzipped folder.

What You Get
